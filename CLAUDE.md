# Project: Personal Portfolio Website

A Next.js + Tailwind CSS personal portfolio site.

## Stack
- Next.js 14+ (App Router)
- React 18
- Tailwind CSS
- TypeScript

## Sections
- Hero — name, role, tagline, CTA
- About — bio
- Projects — project cards with links
- Contact — email + social links

## Commands
- `npm run dev` — start dev server (http://localhost:3000)
- `npm run build` — production build
- `npm run start` — start production server
- `npm run lint` — lint

## Conventions
- App Router pages live in `app/`.
- Reusable components live in `components/`.
- Content (projects, bio, links) lives in `content/site.ts` so it is easy to edit.
